# India_Savings_Awareness_Dashboard_Project
The India Savings Awareness Dashboard is an interactive Microsoft Excel dashboard designed to analyze the financial saving habits of Indians and promote awareness about investment options like SIP (Systematic Investment Plan) and SWP (Systematic Withdrawal Plan).

This project combines data analysis, dashboard design, financial insights, and Excel automation to transform raw survey data into meaningful business insights that can help individuals, financial advisors, and organizations understand savings behavior and encourage better financial planning.
